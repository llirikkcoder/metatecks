# Старые VPS конфиги

Эта папка содержит конфигурационные файлы для развертывания на VPS (устаревшие).

## Файлы:

- `nginx.conf` - Nginx конфиг с SSL и uwsgi
- `uwsgi.ini` - uWSGI конфиг
- `supervisor.conf` - Supervisor конфиг для управления процессами
- `mirror_nginx.conf` - Зеркальный nginx конфиг

## Важно:

Эти конфиги **НЕ используются** в Docker-окружении.

Для Docker используйте:
- `docker/nginx/` - конфигурация nginx для Docker
- `docker-compose.yml` - основная конфигурация Docker
- `.env.docker` - переменные окружения

Эти файлы сохранены для справки и обратной совместимости с VPS.
